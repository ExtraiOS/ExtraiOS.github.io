BarMagnet

Install
Description

BarMagnet is a powerful torrent remote controller for iOS, gorgeously designed for your iDevice. This app can connect to a multitude of torrenting applications. It currently supports Deluge, qBittorrent, ruTorrent (HTTPRPC plugin), rTorrent (XMLRPC), SeedStuff, Synology, Transmission, Vuze, and uTorrent. It also includes a query system that lets you easily search any torrent website that inserts the query into the URL, while also letting you add extra modifiers to automatically sort by whichever parameter you desire.
